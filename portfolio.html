Restaurant & Takeaway

Johnny's Pizza

One-Page Website Restaurant Responsive Design

The Challenge

Johnny's Pizza on Lisleby in Fredrikstad had been serving the community since 1998, but relied entirely on Foodora for online orders. They paid 6.5% + fees per order — a cost that ate into their margins. They needed an independent web presence that would showcase their menu, build their brand, and give customers a direct way to order.

Our Solution

We built a vibrant one-page website that captures the warmth and personality of this family-run pizzeria. The site features a full menu section, photo gallery with real food images, customer reviews, delivery info with pricing, and a business case page showing the financial benefit of direct orders. The design is warm, inviting, and mobile-first — perfect for hungry customers on the go.

Key Features

  • Responsive one-page design
  • Interactive menu with categories
  • Photo gallery integration
  • Customer review display
  • Delivery zone & pricing info
  • Direct call & Foodora/Wolt links
  • Business case calculator

Tech Stack

HTML/CSS JavaScript Responsive Design Netlify

Agency Website

DevForge — This Site

Multi-Page Agency Advanced SEO

The Challenge

As a new web development agency, we needed a site that would establish credibility, showcase our skills, and generate leads — all without expensive templates or build tools. The site had to demonstrate our technical capabilities while being honest about our experience level.

Our Solution

We built this 6-page dark-themed website from scratch using pure HTML, CSS, and JavaScript — zero dependencies, no build tools. The design features indigo and cyan gradients, smooth scroll animations, and a fully responsive layout. We implemented advanced technical SEO including JSON-LD structured data (Organization, LocalBusiness, Service, FAQ, BreadcrumbList), Open Graph tags, and an llms.txt file for AI crawlers.

Key Features

  • 6 custom-designed pages
  • Dark theme with gradient accents
  • Responsive mobile-first design
  • JSON-LD structured data (5 schema types)
  • AI-ready (robots.txt + llms.txt)
  • Security headers via Netlify
  • Accessibility (skip-link, ARIA, reduced motion)
  • Netlify Forms contact integration

Tech Stack

HTML/CSS JavaScript JSON-LD Netlify

Local Boutique Store

E-Commerce Concept Stripe

The Challenge

Many local retail stores in Norway want to sell online but are overwhelmed by the complexity of e-commerce platforms. They need a simple, clean solution that lets them list products, accept payments, and manage orders — without a steep learning curve or expensive monthly subscriptions.

Our Solution

We designed a concept e-commerce store for a local boutique. The site features product listings with categories, a shopping cart, Stripe payment integration, order management, and responsive design. This concept demonstrates our ability to build full e-commerce solutions from $3,000.

Key Features

  • Product catalog with categories
  • Shopping cart functionality
  • Stripe payment integration
  • Responsive design
  • Order management dashboard
  • Inventory tracking
